Các thẻ cơ bản - hiển thị hình ảnh, liên kết

Hiển thị hình ảnh: img (src, alt,...)

        thẻ img:
            - thuộc tính: src = [url] (nguồn ảnh)
                url: tương đối / tuyệt đối
                + url: tuyệt đối 
                    [giao thức]://domain/path
                    ví dụ:
                    https://devmaster.edu.vn/images/logo.png
                + url: tương đối
                    ++ folder/file; images/logo.png
                    ++ file : logo.png
                    ++ ../ : ../logo.png
